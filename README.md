# base-image

## 镜像介绍
mpibuilder_base: MPI应用程序编译与构建环境，包含完整MPICH和编译指令

mpirun_base: MPI应用程序运行环境，仅包含mpirun和基础动态链接

## 镜像构建
先构建 mpibuilder_base ，成功后再构建 mpirun_base
